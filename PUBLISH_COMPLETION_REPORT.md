```bash

```

**Personal Access Token创建步骤:**
1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token"
3. 选择 "repo" 权限
4. 复制生成的token
5. 在推送时使用token作为密码

### 2. 配置GitHub Actions
